Offline HYPERNETS API
======================

The `OfflineHYPERNETSAPI` class provides access to local HYPERNETS archive data.
It supports filename lookups and SQLite query execution against an on-disk archive database.

The offline API is typically instantiated with the archive root folder and archive database name.
